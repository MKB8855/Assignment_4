# Data Quality Assessment & Preprocessing - Wine Quality Dataset

## Lab Overview
This lab focuses on performing data cleaning and preprocessing on the **Wine Quality Dataset (WineQT)**. The goal is to prepare the data for machine learning by addressing data quality issues, handling outliers, and applying dimensionality reduction.


## Tasks Implemented

### Task 1: Data Quality Assessment
- Inspected data types and ensured all features are in the correct numerical format.
- Checked for initial missing values.

### Task 2: Missing Value Strategy
- Introduced artificial missing values for educational purposes.

### Task 3: Outlier Detection & Handling
- Visualized outliers using **Boxplots** for key features like `alcohol` and `residual sugar`.
- Used the **Interquartile Range (IQR)** method to detect and remove extreme outliers to improve data consistency.

### Task 4: Normalization & Scaling
- Applied **Min-Max Normalization** to scale features between 0 and 1.
- Applied **Z-score Standardization** to transform data to have a mean of 0 and a standard deviation of 1.

### Task 5: Principal Component Analysis (PCA)
- Applied **PCA** to reduce the dataset's dimensionality into 2 Principal Components (PC1, PC2) while retaining significant variance.
